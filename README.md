# 지금 뜨는 키워드 (1단계)

한국에서 지금 급상승 중인 검색 키워드를, 1시간 단위 흐름으로 보는 개인용 트렌드 대시보드.
구글 트렌드(한국) RSS만 사용하므로 **API 키 발급도, 비용도 없습니다.**

## 무엇이 되나
- 지금 한국에서 뜨는 키워드 상위 10개 + 각 키워드의 관련 뉴스 한 줄
- 신규 진입 / 3시간 이상 유지 키워드 개수 요약
- 최근 6시간 동안 키워드 순위가 어떻게 움직였는지 보여주는 흐름 그래프
- GitHub Actions가 **1시간마다 자동으로** 데이터를 수집·누적 → 안 봐도 24시간치가 쌓임

## 파일 구성
```
trend-now/
├── index.html              화면 (토스 스타일)
├── fetch_trends.py         구글 트렌드 수집 스크립트
├── data/
│   ├── latest.json         최신 스냅샷
│   └── history.json        24시간 누적 기록
└── .github/workflows/
    └── collect.yml         1시간마다 자동 실행
```

## 설치 (10분, 무료)

1. **GitHub에 새 저장소(public)를 만든다.** 예: `trend-now`
   - 공개(public) 저장소여야 GitHub Actions가 무료입니다.

2. **이 폴더의 파일을 전부 업로드한다.**
   - `data/` 안의 json은 예시 데이터입니다. 자동 수집이 한 번 돌면 실제 데이터로 덮어써집니다.

3. **GitHub Pages를 켠다.**
   - 저장소 Settings → Pages → Source를 `main` 브랜치 / `/ (root)` 로 설정 → Save
   - 잠시 후 `https://(내아이디).github.io/trend-now/` 주소가 생깁니다.

4. **Actions를 한 번 수동 실행한다.**
   - 저장소 Actions 탭 → "트렌드 수집 (1시간마다)" → Run workflow
   - 처음 한 번 돌리면 실제 데이터가 채워지고, 이후로는 매시 정각마다 자동으로 갱신됩니다.

> 참고: GitHub의 schedule은 정확히 정각이 아닐 수 있고 부하에 따라 몇 분 늦을 수 있습니다.
> 1시간 단위 모니터링에는 문제가 없습니다.
